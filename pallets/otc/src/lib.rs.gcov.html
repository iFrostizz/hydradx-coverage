      16             : // # OTC pallet
      17             : // ## General description
      18             : // This pallet provides basic over-the-counter (OTC) trading functionality.
      19             : // It allows anyone to `place_order` by specifying a pair of assets (in and out), their respective amounts, and
      20             : // whether the order is partially fillable. The order price is static and calculated as `amount_out / amount_in`.
      21             : //
      22             : // ## Notes
      23             : // The pallet implements a minimum order size as an alternative to storage fees. The amounts of an open order cannot
      24             : // be lower than the existential deposit for the respective asset, multiplied by `ExistentialDepositMultiplier`.
      25             : // This is validated at `place_order` but also at `partial_fill_order` - meaning that a user cannot leave dust amounts
      26             : // below the defined threshold after filling an order (instead they should fill the order completely).
      27             : //
      28             : // ## Dispatachable functions
      29             : // * `place_order` -  create a new OTC order.
      30             : // * `partial_fill_order` - fill an OTC order (partially).
      31             : // * `fill_order` - fill an OTC order (completely).
      32             : // * `cancel_order` - cancel an open OTC order.
